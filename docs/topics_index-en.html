Research Topics

Theory of Ontology

This research project is concerned with basic theory of ontology, ontology representation language, and tool which play important roles in ontological engineering.

Current research topics include;
  • Basic theories of ontology
  • Environment for building ontology
  • Task ontology representation language
  • End-user programming environment: CLEPE

Kowledge-level Communication

  • デュアルループモデルに基づく知識マネジメント支援
  • オントロジーに基づいた協調型合意形成支援
  • Task ontology based authoring system for training systems
  • Framework for group learning: FITS/CSCL
  • General framework for ITS: FITS
  • Nonmonotonic inductive student modeling algorithm

Knowledge Systematization

  • Systematization of Education Knowledge
  • Functional representation and functional understanding
  • Systematization of Nanotechnology Knowledge
