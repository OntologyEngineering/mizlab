オントロジー基礎・言語・ツール

この研究プロジェクトでは，オントロジー工学を支える基礎理論，言語，ツールに関する研究を行っている。基本的な目標は，

  1. 安定した信頼性の高いモデル構築を支える基礎理論を明確にし，
  2. モデル構築を含めた運用を視野に入れた理論体系を明らかにする

ことである。現在，以下のテーマについて研究を行っている。

  • オントロジー基礎論
  • オントロジー構築・利用環境「法造」
  • タスクオントロジー記述言語
  • エンドユーザプログラミング環境：CLEPE

オントロジーの哲学的考察

  • 遠心力の存在証明
