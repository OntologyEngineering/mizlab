Ontology Building and Useing Environment

This is an under-development version of an Ontology Building and Useing Environment. (Version 1.0)

Notes:
  • Don't close any windows by your window manager (i.e., Don't push the close button of the windows). In order to stop the program, just quit your browser.
  • Don't use Japanese letters.

Platforms:

  • Windows95:
    • Microsoft Internet Explorer (IE) 3.01 (3.0 has many bugs)
    • Netscape Navigator 2.02 (3.0 does not work!)
  • Macintosh:
    • Apple Applet Runner in MacOS Runtime for Java (MRJ) 1.0
    • Netscape Navigator 3.0 works, but the system frequently hungs up.
  • UNIX:
    • Netscape Navigator
