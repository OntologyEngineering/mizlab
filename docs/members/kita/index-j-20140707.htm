研究内容

主に人工物に関する知識のモデル化とそれらの共有支援に興味があります．基本的目標は，

  1. 人間のナイーブな理解を支える基本的概念の成り立ちを明らかにし，
  2. 明示化されたそれに基づいて工学的知識の記述を支援し，
  3. その知識を技術者間で流通させたり，工学的問題解決を支援する計算機システムを実現する

ことです．現在，以下のような概念とその扱いについて考察を行っています．
