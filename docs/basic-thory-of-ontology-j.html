オントロジー基礎論

オントロジー研究は世界の成り立ちを説明する哲学の試みから，具体的な対象の情報モデルを構築する情報科学の試みまで広いスペクトルを持 つ．しかし，安定した信頼性の高いモデル構築を目指すとき，両者に共通して言えることは確固たる基礎理論の必要性である．中でも，事物の全体性を構成する 原理を考察する全体・部分の理論，同一性の生起と消滅を論じるidentityの理論そして，事物間の本質的な相互依存関係を考察する依存関係の理論はオ ントロジー構築の基礎理論を構成する．これらの理論はモデリングにおいて頻繁に用いられる，is-a，part-of，instance-ofなどの基本 意味リンクの「意味」の明確な定義と運用に本質的に関わっており，基礎から具体的なモデル構築までの広い範囲に渡って重要な役割を演じる．本研究課題で は，３つの基礎理論の考察を深めながら「意味リンクの意味論」の確立を目指して研究を進めている．

現在考察を進めている問題は以下の通りである．

  1. instance-of と member-of（クラスと集合）の相違は何か？
  2. is-a と part-of の相違は何か？
  3. メタ・オブジェクト関係と上位・下位関係（instance-of と is-a）の相違点は何か？
  4. part-of リンクは何種類必要か？
  5. 関係のインスタンスとは何か？
  6. 概念「妻」はインスタンスを持つことができない？
